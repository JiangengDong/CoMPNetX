baxter_common Repository Overview
---------------------------------

::

     .
     |
     +-- baxter_common/           baxter_common metapackage
     |
     +-- baxter_description/      urdf and meshes describing baxter
     |   +-- urdf/
     |   +-- meshes/
     |
     +-- baxter_core_msgs/        messages and services for communication with baxter
